# MathOptAI (Mo'ai)

_If you can come up with a better name, please open an issue._

MathOptAI.jl (Mo'ai) is a [JuMP](https://jump.dev) extension for embedding
trained AI, machine learning, and statistical learning models into a JuMP
optimization model.

time._
